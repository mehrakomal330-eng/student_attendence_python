#  Python Attendance System

A simple attendance management project using **Python** and **Tkinter** that records student attendance, analyzes results, and visualizes data.

##  Features
- Mark and save attendance  
- Show attendance percentage  
- Display bar & pie charts  
- Simple Tkinter interface  

##  Technologies
Python | Tkinter | Matplotlib | CSV

##  How to Run
1. Clone the repo  
   `git clone https://github.com/yourusername/attendance-system.git`  
2. Run  
   `python attendance_system.py`

##  Output
- Attendance Entry Window  
- Attendance Charts (Bar & Pie)
